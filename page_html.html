What is HTML?

HTML is the standard markup language for creating web pages. It provides the structure and content of websites by using a system of tags and elements. Every website you see starts with HTML!

Basic HTML Structure

<!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>My First Website</title>
                        </head>
                        <body>
                            <h1>Welcome to HTML!</h1>
                            <p>This is my first paragraph.</p>
                        </body>
                        </html>
