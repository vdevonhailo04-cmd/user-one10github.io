(PHP: Hypertext Preprocessor)

Server-side scripting language for web development

What is PHP?

PHP is a server-side scripting language used to create dynamic web pages. It's executed on the server and the result is sent to the user's browser as HTML.

PHP is one of the most popular server-side languages, powering millions of websites including WordPress, Facebook, and Wikipedia. It's easy to learn and great for web development.

PHP Basics

<?php
    // PHP code goes here
    echo "Hello, World!";
    
    // Variables
    $name = "John";
    $age = 25;
    
    // Functions
    function greet($person) {
        return "Hello, " . $person;
    }
    
    // Arrays
    $fruits = array("Apple", "Banana", "Orange");
    
    // Conditional
    if ($age > 18) {
        echo "You are an adult";
    }
?>

PHP Concepts

  • Variables: Start with $ symbol - $name = "John";
  • Functions: Reusable code blocks - function myFunc() { }
  • Arrays: Indexed or associative arrays for storing data
  • Form Handling: Process GET and POST requests from HTML forms
  • Database: Connect to MySQL or other databases with PDO/MySQLi
  • Sessions: Maintain user state across pages with $_SESSION
  • Cookies: Store data on client side with setcookie()
  • Object-Oriented PHP: Classes and objects for advanced programming

Superglobals

$_GET

Get URL parameters from query string

Example: $_GET['name']
$_POST

Get form data from POST requests

Example: $_POST['username']
$_SESSION

Store session variables for user

Example: $_SESSION['user_id']
$_COOKIE

Access cookie data from browser

Example: $_COOKIE['remember']
$_SERVER

Server and execution environment info

Example: $_SERVER['HTTP_HOST']
$_FILES

Access uploaded files from forms

Example: $_FILES['upload']['name']
