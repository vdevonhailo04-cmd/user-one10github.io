Object-oriented programming for robust applications

What is Java?

Java is a general-purpose, object-oriented programming language. It's known for its "write once, run anywhere" capability and is widely used in enterprise applications.

Java runs on the Java Virtual Machine (JVM), which allows the same code to run on any platform that has a JVM installed. This makes Java incredibly portable and flexible.

Java Basics

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// Variables
int age = 25;
String name = "John";
double price = 19.99;

// Arrays
int[] numbers = {1, 2, 3, 4, 5};

// Loops
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Java Concepts

  • Classes: Blueprint for objects - class MyClass { }
  • Objects: Instances of classes created with new keyword
  • Inheritance: Extend classes - class Child extends Parent { }
  • Polymorphism: Multiple forms of methods and method overriding
  • Encapsulation: Hide internal details with private/public keywords
  • Collections: ArrayList, HashMap, HashSet, LinkedList
  • Exception Handling: Try-catch-finally blocks for error handling
  • Multithreading: Concurrent execution with Thread class

Java Data Types

byte

8-bit integer (-128 to 127)

short

16-bit integer (-32,768 to 32,767)

int

32-bit integer (most common)

long

64-bit integer for large numbers

float

32-bit decimal number

double

64-bit decimal (more precise)
