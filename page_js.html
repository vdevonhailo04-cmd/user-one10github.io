What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It's an essential part of web applications and can be used on both client-side and server-side (Node.js).

JavaScript runs in the browser and allows you to manipulate HTML, respond to user actions, and create dynamic content without requiring a page reload.

JavaScript Basics

// Variables
let name = "John";
const age = 25;
var city = "New York";

// Functions
function greet(person) {
    return "Hello, " + person;
}

// Arrays
let fruits = ["apple", "banana", "orange"];

// Objects
let person = {
    name: "Alice",
    age: 30,
    city: "Paris"
};

// Conditional
if (age > 18) {
    console.log("Adult");
}

JavaScript Concepts

  • Variables: Store data using let, const, or var
  • Data Types: String, Number, Boolean, Array, Object, null, undefined
  • Functions: Reusable blocks of code - function myFunc() { }
  • DOM Manipulation: Change HTML content dynamically using DOM methods
  • Event Handling: Respond to user interactions (click, submit, etc.)
  • Async/Await: Handle asynchronous operations cleanly
  • Promises: Handle operations that will complete later
  • ES6+: Modern JavaScript features (arrow functions, destructuring, etc.)

Common Methods

getElementById()

Select element by ID

Example: document.getElementById("id")
querySelector()

Select element by selector

Example: document.querySelector(".class")
addEventListener()

Attach event listener

Example: element.addEventListener('click', func)
setTimeout()

Execute after delay in milliseconds

Example: setTimeout(func, 1000)
forEach()

Loop through array elements

Example: arr.forEach(item => {...})
fetch()

Make HTTP requests

Example: fetch(url).then(res => res.json())
