What is CSS?

CSS is used to style and layout web pages. With CSS, you can control colors, fonts, spacing, and much more to make your websites look beautiful and professional.

CSS works by selecting HTML elements and applying visual styles to them. Think of it as the "paint and design" layer of your website, while HTML is the structure.

Three Ways to Use CSS

/* 1. External CSS (Recommended) */
<link rel="stylesheet" href="style.css">

/* 2. Internal CSS */
<style>
    body {
        background-color: blue;
    }
</style>

/* 3. Inline CSS */
<h1 style="color: red;">Hello</h1>

CSS Selectors

Selectors are patterns used to select the elements you want to style.

Element Selector

p { color: blue; }

Selects all <p> elements
Class Selector

.classname { color: red; }

Selects elements with class
ID Selector

#idname { color: green; }

Selects specific element by ID
Attribute Selector

input[type="text"] { }

Selects by HTML attribute
Pseudo-class

a:hover { color: orange; }

Selects element in state
Descendant Selector

div p { color: purple; }

Selects nested elements

Common CSS Properties

  • color: Text color - color: #FF6B6B;
  • background-color: Background color - background-color: lightblue;
  • font-size: Size of text - font-size: 16px;
  • margin: Space outside elements - margin: 20px;
  • padding: Space inside elements - padding: 15px;
  • border: Element border - border: 2px solid black;
  • display: How element is displayed - display: flex;
  • width & height: Dimensions - width: 100%; height: 200px;
  • box-shadow: Add shadow effects - box-shadow: 0 4px 6px rgba(0,0,0,0.1);
  • border-radius: Round corners - border-radius: 8px;
