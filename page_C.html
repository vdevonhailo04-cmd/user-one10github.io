What is C#?

C# is a modern, object-oriented programming language developed by Microsoft. It's used to build Windows applications, web applications (ASP.NET), and games (Unity).

C# combines the power of C++ with the simplicity of Visual Basic. It runs on the .NET Framework and provides excellent tools for professional development.

C# Basics

using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}

// Variables
int age = 25;
string name = "John";
double price = 19.99;
bool isActive = true;

// Methods
public int Add(int a, int b) {
    return a + b;
}

// Arrays
int[] numbers = new int[] {1, 2, 3, 4, 5};

C# Concepts

  • Classes & Objects: OOP foundation for C# - class MyClass { }
  • Properties & Methods: Class members and behavior
  • Inheritance: Derive from base classes - class Child : Parent { }
  • Interfaces: Define contracts - interface IShape { }
  • Generics: Type-safe collections - List<T>
  • LINQ: Query language for data - Language Integrated Query
  • Async/Await: Asynchronous programming - async Task Method()
  • Delegates & Events: Event handling and callbacks
